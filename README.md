# Python Lessons

A personal repository for teaching basic Python principles.

## Overview

This repo contains simple examples and exercises focused on core Python concepts, including:

- Variables and data types
- Strings and numbers
- Conditionals
- Loops
- Functions
- Collections
- Modules
- Basic file handling

## Purpose

The goal of this project is to provide clear, beginner-friendly material for learning Python step by step.

## Usage

Each lesson can be explored independently.  
Read the examples, run the code, and modify it to practice.

## Structure

The repository is organized into small lessons and exercises to keep topics easy to follow.

## Notes

This project is intended for personal teaching and learning purposes.

## Getting Started

### Clone the Repository

**Windows (Command Prompt or PowerShell):**
```bash
git clone https://github.com/yourusername/python_lessons.git
cd python_lessons
```

**Linux/macOS (Terminal):**
```bash
git clone https://github.com/yourusername/python_lessons.git
cd python_lessons
```

Once cloned, you can start exploring the lessons and running the Python scripts.
